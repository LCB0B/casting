# Casting

A single-file browser game about casting fashion shows — and about what the runway's
idea of beauty costs. You name a house and, season after season, swipe through an endless
casting call to book five looks per show. Four hidden panels — the Board, All Walks, the
Census, and the Front Row — judge every lineup and decide whether the house thrives or
fires you.

The thin-white-blonde "ideal" dominates the stream, so building a genuinely diverse show
takes deliberate work. Tokenism (one body carrying every quota) is the cheap path;
spreading representation across distinct bodies is the hard one. That friction is the point.

## Play

Play online: **https://lcb0b.github.io/casting/**

Or open `index.html` in any modern browser. No build step, no dependencies — everything
(art, animation, logic) lives in the one file.

## Tech

- Pure HTML/CSS/JS, no frameworks or network calls
- Inline SVG figures and gauges, CSS animations
- Mobile-first, swipeable card deck
